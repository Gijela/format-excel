# Excel 运算工具

一个简洁的 React Web 应用，用于对 Excel 文件进行列运算和排序。

## ✨ 功能特性

- 📤 **拖拽上传** - 支持拖拽或点击上传 Excel 文件
- 🔢 **灵活公式** - 支持 `(列A ± */ 列B) ± */ 列C` 形式的运算
- 📊 **自动排序** - 根据运算结果升序或降序排列
- 💾 **规则复用** - 公式配置自动保存，新文件自动应用
- 📥 **一键导出** - 导出包含运算结果的 Excel 文件

## 🚀 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

## 🛠️ 技术栈

- React 18
- Vite 5
- Tailwind CSS
- xlsx (SheetJS)

## 📖 使用说明

1. 拖拽 Excel 文件到上传区域
2. 选择参与运算的三个列
3. 配置运算符（+、-、*、/）
4. 选择排序方式（升序/降序）
5. 点击"执行运算并排序"
6. 查看结果并导出

## 📋 公式格式

```
(列A [运算符1] 列B) [运算符2] 列C
```

例如：`(销售额 * 数量) / 成本`
